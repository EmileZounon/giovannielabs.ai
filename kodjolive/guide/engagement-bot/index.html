Automatisation d'engagement · pret en 45 minutes

Ramenez vos etudiants silencieux.

Des qu'un etudiant ne dit plus rien depuis plus de 48 heures, le bot lui envoie un message chaleureux et personnalise, ecrit par Claude. Pas de template. Pas de relance froide. Juste un petit mot bienveillant qui le ramene avant qu'il ne decroche pour de bon. Telegram d'abord, email en fallback automatique.

Ce que ca fait

Le bot en 4 etapes

01

Ecoute le groupe

Chaque fois qu'un etudiant ecrit dans le groupe Telegram, le bot enregistre qui, quand, et quoi dans un Google Sheet. Silencieux, en arriere-plan.

02

Verifie a 9h du matin

Tous les jours, il parcourt la liste des etudiants et identifie ceux qui n'ont pas ecrit depuis 48 heures ou plus.

03

Claude genere un message

Pour chaque etudiant silencieux, Claude ecrit un petit mot personnalise en francais, dans un ton doux et bienveillant. Deux phrases, un prenom, une question legere.

04

Envoie en cascade

Tentative Telegram DM en premier (gratuit, personnel). Si l'etudiant n'a pas active le bot, bascule automatique sur email. Chaque envoi est loggue pour votre dashboard.

Les 7 etapes du deploiement

Ordre strict : sautez une etape et le reste ne marchera pas.

  1. Creer le bot Telegram

    Sur Telegram, cherchez @BotFather. /newbot, donnez un nom et un username. Copiez le token. Tres important : /setprivacy → Disable (sinon le bot ne voit pas les messages normaux).

  2. Ajouter le bot au groupe comme admin

    Parametres du groupe → Administrateurs → Ajouter le bot. Sans le statut admin, privacy mode off ne suffit pas.

  3. Creer le Google Sheet

    Nouveau Sheet, 3 onglets nommes exactement students, messages, outreach_log. Importez les CSV templates ci-dessus pour avoir les en-tetes. Copiez l'ID du Sheet depuis l'URL.

  4. Recuperer une cle API Anthropic

    Sur console.anthropic.com → API Keys → Create Key. Copiez et gardez-la en securite.

  5. Importer le workflow dans n8n

    n8n Cloud → Workflows → Import from File → choisir kodjo-engagement-bot.json. Le workflow s'ouvre avec 12 nodes visibles.

  6. Brancher les credentials (4)

    Dans chaque node concerne : Telegram (token), Google Sheets (OAuth), Gmail (OAuth), Anthropic (cle). Dans les 4 nodes Google Sheets, remplacez REPLACE_WITH_GOOGLE_SHEET_ID par votre ID.

  7. Activer le workflow et tester

    Toggle Active en haut a droite. Suivez le guide de test pour valider les deux flows avant de laisser tourner.

Cout estime : n8n Cloud (deja paye) + environ 5 $ par mois de Claude API pour un groupe de 20 etudiants actifs. Email gratuit via Gmail. Telegram gratuit.
Points d'attention

Ce qu'il faut savoir avant de lancer

Consentement des etudiants

Avant d'activer le bot, postez ce message dans le groupe :

Pour mieux vous accompagner, notre bot suit votre participation dans le groupe. Si vous n'ecrivez pas pendant 48 heures, vous recevrez un petit mot amical en DM ou par email. C'est juste pour prendre de vos nouvelles. Repondez STOP si vous ne voulez plus recevoir ces rappels.

Activation des DM Telegram

Telegram impose une regle : un bot ne peut DM un etudiant que si celui-ci a clique /start sur le bot au moins une fois. Ceux qui ne le font pas recevront leurs messages par email (fallback automatique).

WhatsApp pour plus tard (v2)

WhatsApp n'est pas inclus dans la v1 car l'API Meta demande une verification business qui prend environ 5 jours ouvres. Une fois la v1 en place et validee, WhatsApp s'ajoute comme un node supplementaire dans la cascade, sans refaire l'architecture.

Vie privee des messages

Le bot enregistre le texte de chaque message du groupe dans un Google Sheet prive. Seul le compte Google du proprietaire du Sheet y a acces. Supprimez les anciennes lignes regulierement si vous voulez limiter l'historique stocke.
