Notre méthode

Le cycle Apprendre · Appliquer · Partager

Chaque cours dure trois semaines. Vous n'absorbez pas seulement de la théorie : vous l'essayez en classe et vous partagez ce que vous avez appris avec vos pairs.

SEMAINE 1

Apprendre

Une session live ou enregistrée. Une vidéo courte. Un modèle téléchargeable. Un quiz pour vérifier votre compréhension.

  • ✓ Session interactive avec facilitateur
  • ✓ Modèle de planification téléchargeable
  • ✓ Quiz de réflexion (5 minutes)
SEMAINE 2

Appliquer

Vous utilisez la technique avec vos vrais élèves cette semaine. Vous documentez ce qui fonctionne et ce qui résiste.

  • ✓ Application concrète en classe
  • ✓ Modèle d'observation guidée
  • ✓ Exemples vidéo de salles de classe réelles
SEMAINE 3

Partager

Session communautaire de 90 minutes. Vous racontez votre semaine, vous écoutez vos pairs, vous recevez un retour personnalisé.

  • ✓ Session live de 90 minutes
  • ✓ Retour personnalisé du facilitateur
  • ✓ Certificat numérique téléchargeable

Comment fonctionne le cycle de 3 semaines ? +

Semaine 1 : vous apprenez la technique. Semaine 2 : vous l'appliquez avec vos élèves. Semaine 3 : vous partagez vos observations avec la communauté.

Et si je manque une session live ? +

Toutes les sessions sont enregistrées. Vous pouvez les revoir à tout moment, même hors ligne sur votre téléphone.

Comment puis-je payer depuis Kinshasa ? +

Mobile Money (Mpesa, Airtel Money, Moov Africa) ou virement bancaire pour les écoles. Le paiement est confirmé instantanément.

Comment fonctionne le tirage au sort hebdomadaire ? +

Vous gagnez des entrées en participant : sessions, réflexions, contributions au forum. Tirage chaque vendredi, gagnants annoncés par SMS et WhatsApp.

Le certificat est-il reconnu ? +

Oui. Chaque certificat porte un code QR unique vérifiable sur le site Koyekola. Les inspecteurs et directeurs peuvent vérifier votre progression.
